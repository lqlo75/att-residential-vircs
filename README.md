## 什么是美国AT&T住宅IP，跟机房IP有什么区别

**美国AT&T住宅IP**，指的是由美国运营商 AT&T 分配给真实家庭用户的宽带 IP 地址，归属信息显示为住宅（Residential）类型，而非数据中心（Datacenter）类型。

这个区别决定了一件很关键的事：TikTok、Instagram、PayPal、Chase 银行这类对账号环境敏感的平台，风控系统会主动识别 IP 来源。机房 IP 属于商用线路，被标记为"高风险"的概率远高于住宅 IP。这也是为什么用同样的配置，有人账号跑得好好的，有人天天被限流、封号。

简单说：住宅IP让你看起来像一个住在美国的普通人在上网。机房IP让你看起来像一台服务器在访问平台。
